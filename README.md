LifeLedger is a Java Swing desktop application built for COP 4331 (Object-Oriented Design). It allows users to:

- Track income and expense transactions
- View total balances across accounts
- Extend to budgets, reports, and reminders

## Technologies
- Java 17
- Swing GUI
- MVC Architecture
- JUnit 5
- Design Patterns (Observer, Strategy, Command, Singleton, Repository)
