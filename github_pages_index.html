LifeLedger – Personal Finance Manager

Course: COP 4331 – Object-Oriented Design

LifeLedger is a Java Swing desktop application that helps users track accounts, transactions, budgets, and spending reports. It showcases MVC architecture, UML-driven design, and multiple design patterns.

Features

  • Manage multiple financial accounts
  • Record income and expense transactions
  • View total balance across accounts
  • Extensible design for budgets, reports, and reminders

Technologies

  • Java 17
  • Swing GUI
  • JUnit 5
  • Design patterns (MVC, Observer, Strategy, Command, Singleton, Repository)
