LifeLedger – Personal Finance Manager

COP 4331 – Object-Oriented Design

Author: Charlie Holder

Project Overview

LifeLedger is a Java Swing desktop application that allows users to manage financial accounts, track income and expenses, view total balances, define budgets, generate reports, and manage bill reminders. The system was designed using full Object-Oriented Design, UML modeling, MVC architecture, and multiple design patterns.

Key Features

  • Create and manage multiple financial accounts
  • Add income and expense transactions
  • Automatic balance updates
  • Category-based reporting and summaries
  • Bill reminders and future extensibility

Technologies Used

  • Java 17
  • Java Swing (GUI)
  • MVC Architecture
  • JUnit 5
  • Design Patterns:
    • Observer
    • Strategy
    • Command
    • Singleton
    • Repository (DAO)

System Architecture

LifeLedger follows the Model–View–Controller (MVC) architecture:

  • Model: Account, Transaction, Budget, Reminder, FinanceModel
  • View: Java Swing GUI (MainFrame and panels)
  • Controller: MainController, TransactionController, BudgetController
